# Vet Clinic Database
## Getting Started

This repository includes files with plain SQL that can be used to recreate a database:

- Use [schema.sql](./schema.sql) to create all tables.
- Use [data.sql](./data.sql) to populate tables with sample data.
- Check [queries.sql](./queries.sql) for examples of queries that can be run on a newly created database. **Important note: this file might include queries that make changes in the database (e.g., remove records). Use them responsibly!**

## ğŸ’» Getting Started <a name="getting-started"></a>

> Describe how a new developer could make use of your project.

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

  - Basic knowledge of relational database.
  - Install postgreSQL database on your local environment.

### Setup

Clone this repository to your desired folder:

  - Open postgreSQL command shale (```psql```) or GUI interactive application like pgAdmin.
  - Copy SQL query from schema.sql and execute to create database and table.
  - Insert data into table by executing query from queries.sql file.
